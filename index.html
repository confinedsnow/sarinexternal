// setup guide

How to run it.

01

Download Python

You need the latest version of Python to run my external (free and often maintained).
Grab the latest installer from the official Python website:

python.org/downloads ↗ OPEN
02

Install Python

Run the installer. Important: on the first screen, check "Add Python to PATH" before clicking Install. This lets you run Python from anywhere on your system.

03

Download Sarin

Grab the file from the download button below and save it anywhere on your PC — Desktop works fine.

04

Run It

Double-click the .py file, or right-click → Open With → Python. That's it. No installs, no setup, no BS.

python sarin_external_featherweight_edition.py COPY
